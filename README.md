# AI Project Planner Tool

A web-based AI Project Planner Tool built as part of the Software Engineering (SE) course presentation.

## 📁 Project Structure

```
├── Web Testing.html   # Main HTML file
├── web-testing.css    # Stylesheet
├── web-testing.js     # JavaScript logic
└── README.md          # Project documentation
```

## 🚀 Features

- Interactive project planning interface
- Clean, modular codebase (HTML, CSS, JS separated)
- Built for SE Presentation — 4th Semester

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## 📖 Usage

Open `Web Testing.html` in any modern browser to run the application.
